🎓 About My Portfolio Website

This is my personal portfolio website, built with HTML, CSS, and vanilla JavaScript

to highlight my skills, projects, and academic journey. It features a clean hero 

section, a short introduction about me, a visual overview of my technical and soft 

skills, and a curated selection of projects with descriptions, tech stacks, and 

GitHub or live links.

The site also includes an education and experience timeline, a simple contact form

for direct communication, and a fully responsive design with smooth animations for a

modern, polished user experience.
